# Deliveries cohort

Code used to generate a deliveries cohort in HES. See the included readme file for more details. Code lists are mainly from the ECHILD Phenotype Code List Repository (https://code.echild.ac.uk) as documented in the guidance.

R code and this repository written by Matthew Jay, matthew.jay@ucl.ac.uk, based on [Ireland et al. IJPDS;2024;9:2:04](https://ijpds.org/article/view/2404/).
